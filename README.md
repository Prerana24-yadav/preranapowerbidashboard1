# preranapowerbidashboard1
The HR Analytics Dashboard provides key insights into workforce demographics, attrition trends, job satisfaction levels, and education field-wise analysis
The HR Analytics Dashboard was created in Power BI using the above dataset. It includes the following visuals:
- KPIs: Overall Employees, Attrition Count, Attrition Rate, Active Employees, and Average Age
- Department-wise Attrition: Pie chart showing attrition distribution across departments
- Age Group Distribution by Gender: Bar chart comparing male and female employees across age bands
- Job Satisfaction Rate: Table showing satisfaction levels for different job roles
- Education Field-wise Attrition: Bar chart showing attrition by education field
- Attrition Rate by Gender & Age Group: Donut charts showing attrition rates by gender for each age category
4. Tools & Technologies Used
• Microsoft Power BI – For data visualization and dashboard design
• Microsoft Excel – For data storage and preprocessing
• GitHub – For version control and project sharing
5. Key Findings
- The overall attrition rate is around 16%.
- Sales department has the highest attrition compared to R&D and HR.
- Most employees fall within the 25–44 age range.
- Employees with Life Sciences and Medical backgrounds contribute the most to attrition.
- Job satisfaction varies significantly across roles, with Research Scientists showing high satisfaction levels.
